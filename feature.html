Git Branchについて

Gitのブランチは、プロジェクトの独立した開発ラインを作成するための機能です。ブランチを使用することで、メインのコードベースに影響を与えずに新しい機能や修正を試すことができます。

ブランチの作成

新しいブランチを作成するには、以下のコマンドを使用します:

git branch ブランチ名

ブランチの切り替え

別のブランチに切り替えるには、以下のコマンドを使用します:

git checkout ブランチ名

ブランチのマージ

ブランチをマージするには、以下のコマンドを使用します:

git merge ブランチ名

新しい機能の追加

新しい機能を追加するには、まず新しいブランチを作成し、そのブランチで開発を行います。開発が完了したら、メインブランチにマージします。

git branch 新しい機能ブランチ名
git checkout 新しい機能ブランチ名
# 新しい機能を開発する
git checkout main
git merge 新しい機能ブランチ名
